# ashish-website

A modern, premium personal website built with HTML, CSS, and JavaScript.

## Features

- 🎨 **Stunning Dark Theme** — Glassmorphism, gradients, and vibrant color palette
- ✨ **Micro-Animations** — Particle background, typing effect, scroll reveals, floating cards
- 📱 **Fully Responsive** — Optimized for desktop, tablet, and mobile
- ⚡ **Zero Dependencies** — Pure HTML, CSS, and JavaScript
- 🧭 **Smooth Navigation** — Sticky navbar with active section tracking

## Sections

- **Hero** — Animated intro with typing effect and stats
- **About** — Personal bio with interactive code window
- **Projects** — Featured project cards with hover effects
- **Skills** — Categorized technology toolkit
- **Contact** — Contact links and form

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/ashishfernandez/ashish-website.git
   ```
2. Open `index.html` in your browser
